## Description

Task Api with ExpressJs.

## Required Before Running

```
- nodejs
- mongodb
```

## Clone Repository

```
git clone https://github.com/jrsmth97/taskapp-express.git .
```

## Environment Prepare

```
copy .env.example and rename to .env
set your mongodburl
```

## Installation

```bash
$ npm install
```

## Running the app

```bash
$ npm start
```
